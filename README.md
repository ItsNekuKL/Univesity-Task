# 📚 Kumpulan Tugas Kuliah 

Repo ini isinya hasil kerjaan tugas kuliah gw, mayoritas fokus ke struktur data sama algoritma pakai Python. Kalo mau dipake buat referensi silakan, tapi pastiin lu paham logikanya. Jangan asal copas terus plonga-plongo pas ditanya dosen.

## ⚙️ Isi Repo

* **Universal Hashing**: Implementasi fungsi hash dinamis buat ngurangin peluang *collision*. Eksekusi cepat, distribusi data lebih merata.
* **Double Linked List**: Traversal data dua arah. Tiap *node* punya *pointer* ke elemen sebelumnya (`prev`) dan sesudahnya (`next`). Fleksibel buat manipulasi data.
* **Stack & Binary Search**: Implementasi struktur data *Last-In-First-Out* (LIFO) dan algoritma pencarian efisien dengan kompleksitas $O(\log n)$.
* *(tambahin list tugas lu yang lain di sini)*
